# Northern Italy Traveler Companion v042.0

## Editorial dashboard refinement

This release develops the existing v041.0 project rather than adding a placeholder version number.

### Implemented

- Reworked the home hero with a larger editorial title, improved photographic treatment and clearer trip information.
- Rebuilt the route overview into a structured six-stage journey timeline.
- Restyled the Today panel, Journey Command Center, reservation priorities and quick-reference cards as one coherent dashboard system.
- Reorganized destination photography into a responsive magazine-style grid.
- Improved desktop, tablet and phone spacing, hierarchy and card proportions.
- Added current-page highlighting to the persistent companion toolbar.
- Changed the phone toolbar into a full-width bottom app navigation with safe-area support.
- Updated version labels, cache identifiers and offline assets to v042.0.

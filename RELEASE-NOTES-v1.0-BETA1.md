# Northern Italy Traveler’s Companion — v1.0 Beta 1

## Design and UX polish

- Introduced a unified spacing, radius, shadow and content-width system across the application.
- Refined editorial typography, headings, mastheads, dashboard alignment and mobile proportions.
- Standardized card elevation, image behavior, buttons, forms, navigation states and footer treatment.
- Improved the dashboard hero alignment and preserved brighter, more natural photography.
- Added clearer current-page states in both the primary navigation and travel toolbar.

## Accessibility and resilience

- Strengthened keyboard focus indicators and the skip-to-content control.
- Added reduced-motion and increased-contrast support.
- Increased minimum touch-target sizes and improved mobile navigation behavior.
- Added accessible navigation labels, page descriptions and secure handling of new-window links.
- Applied lazy loading and asynchronous decoding to non-critical imagery.

## Offline and release updates

- Updated the PWA manifest, visible version labels and service-worker cache.
- Added this release note to the offline cache.

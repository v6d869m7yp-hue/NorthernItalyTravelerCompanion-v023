# Northern Italy Traveler Companion — v1.0 Beta 1.1

## Sitewide Layout Stabilization Sprint

This release repairs recurring responsive-formatting problems across the complete site rather than patching individual screenshot examples.

### Repairs

- Preserves complete illustrated spreads in heroes, feature panels, galleries and guide cards.
- Removes fixed-height cropping from paired image-and-text sections.
- Constrains oversized hero titles to safe, responsive widths.
- Prevents editorial columns, headings and ledger rows from colliding or beginning offscreen.
- Rebuilds tablet breakpoints so split layouts stack before content overlaps.
- Corrects full-width section gutters and removes clipping caused by inherited overflow rules.
- Hides the floating travel toolbar on desktop and reserves clearance for it on phone/tablet layouts.
- Adds safer intrinsic sizing and overflow wrapping throughout shared layout primitives.

No new travel features were added in this stabilization release.

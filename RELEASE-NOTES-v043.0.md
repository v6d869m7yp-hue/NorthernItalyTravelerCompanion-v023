# v043.0 — Integrated Day Planner

This release connects the existing Today mode and Reservations wallet.

## Implemented
- Today mode now displays reservations saved for the selected travel date.
- Bookings are ordered by time and show status, location, and confirmation number.
- Empty travel days offer a direct date-prefilled reservation link.
- The Reservations form accepts a date passed from Today mode.
- Styling is responsive and remains available offline through the existing local-storage system.
- Version labels and the service-worker cache were advanced to v043.0.

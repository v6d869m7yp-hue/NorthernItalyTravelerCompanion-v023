# Northern Italy Traveler’s Companion — v1.0 Beta 1.3.2

A complete offline-capable travel website for the August 31–September 10 Northern Italy itinerary.

## v1.0 Beta 1.3.2 highlight: Integrated Day Planner

- Today mode displays bookings saved for the selected travel date.
- Same-day bookings are ordered by time and show status, location, and confirmation details.
- Empty days include a direct **Add reservation** action with the correct date already filled.
- The existing Reservations wallet remains private to the browser and available offline.
- Mobile and desktop layouts were updated for the additional day-planning panel.

## Included

- 16 HTML pages plus all local assets
- Destination guides for Venice, the Dolomites, Cinque Terre, Lake Como, Lake Maggiore, and Milan
- Today mode, daily itinerary, My Trip favorites, maps, practical guidance, and Reservations wallet
- PDF-restored destination image library
- Progressive web app manifest and service-worker offline cache

## Installation

Copy everything inside this folder to the root of the GitHub repository. This is a complete project package, not a patch.

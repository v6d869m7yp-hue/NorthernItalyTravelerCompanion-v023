# Northern Italy Traveler’s Companion — Version 026.1

Version 026.1 is a locked-scope responsive correction release built from v026.

## Five responsive fixes
1. Stable header and mobile navigation across window widths.
2. Fluid hero and masthead typography without clipping.
3. Predictable grid and card reflow on tablet and phone widths.
4. Viewport-safe images, maps and reference tables.
5. Consistent narrow-screen spacing, safe-area support and landscape behavior.

No destination content or feature scope was changed.

Deploy the contents of this folder at the repository root.

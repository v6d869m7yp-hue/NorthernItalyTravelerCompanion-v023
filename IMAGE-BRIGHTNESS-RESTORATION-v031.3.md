# v031.3 Image Brightness Restoration

The source photographs remain unchanged. A final CSS correction layer now overrides accumulated darkening from older releases.

Corrected areas:
- Home dashboard hero
- Venice and Lake Como Signature Guide heroes
- Milan hero photography
- Destination and Explorer cards
- Magazine-style covers

The final layer modestly increases displayed brightness and saturation, substantially reduces full-image overlay opacity, and uses localized text shadow for readability instead of darkening the entire photograph.
